mlr3 Exercises Day 3 - Ames Housing Dataset

This exercise is intended to use mlr3pipelines to create machine learning pipelines to improve the results of the benchmark analysis on the "Ames Housing Dataset".
There exist a new Kaggle InClass challenge containing the original dataset, i.e. there are missing values and factor variables in the data to deal with.
You can find this new challange on
http://www.kaggle.com/c/ames-day3.
